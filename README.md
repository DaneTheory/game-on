CORS
====

> CORS authentication sample app built with Yeoman, AngularJS and NodeJS.

> http://pablodenadai.github.io/CORS/
  
##Install:

###Front-end: 

#### Requirements:
  + Yeoman
  + Grunt
  + Bower
  
```$ grunt server```
  
  
###Back-end:

#### Requirements:
  + Node
  
```$ node app.js```
